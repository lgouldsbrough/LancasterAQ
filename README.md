# LancasterAQ Package
Python package developed for the Lancaster Air Quality project.

## Install Instruction
### Manual Installation
```bash
# clone from github
$ git clone <url>
# change directory into project root
$ cd LancasterAQ

# regular install
$ pip install .
# or 
# development install 
# $ pip install -e .
```
